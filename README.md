## ❓ Troubleshooting

- **Discord doesn’t patch:** Make sure Discord is fully closed before starting your PC.
- **Permissions errors:** Run the tool as administrator. Right-click the installer and choose "Run as administrator".
- **Tool does not start with Windows:** Check your Startup apps in Windows Settings.
- **Antivirus blocks the tool:** Your security software might flag the patcher. Add it as an allowed app or temporarily disable antivirus during patching.
